### Installation

![Screenshot](./public/thumbnail-preview.png)

- Clone this repo:

```sh
git clone https://github.com/hebersolano/property-pulse.git
```

- Install dependencies:

```sh
npm install
```

- Build command:

```sh
npm run build
```

- Live server:

```sh
npm run dev
```
